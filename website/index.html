Features

Alles was dein Server
wirklich braucht

🔨

Vollständige Moderation

Ban, Kick, Discord-Timeout, Warn, Clear und Slowmode — mit Rollen-Hierarchie-Prüfung und automatischen Log-Einträgen.

🛡️

AutoMod — Immer aktiv

Kein Einrichten nötig. AutoMod erkennt Schimpfwörter, Invite-Links, Massen-Pings und Caps-Spam vollautomatisch.

🎫

Ticket-System

Button-basierter Support. Tickets erstellen, übernehmen und schließen. Der User wird bei jedem Schritt per DM informiert.

📩

DM-Benachrichtigungen

Bei Mod-Aktionen, Ticket-Ereignissen und Rollen-Änderungen bekommt der User automatisch eine detaillierte DM.

📋

Event-Logging

Join, Leave, Nachrichtenänderungen und Voice-Events werden übersichtlich in einem konfigurierten Log-Channel protokolliert.

👋

Welcome-System

Neue Mitglieder werden im Server begrüßt und erhalten eine persönliche Willkommens-DM mit Server-Informationen.

AutoMod

Automatisch.
Immer aktiv.

AutoMod läuft auf jedem Server — ohne Einrichtung, ohne Aktivierung. Verstöße werden gezählt und lösen automatisch Konsequenzen aus. Admins und Moderatoren sind immer ausgenommen.

🚫
Schimpfwörter
Festgelegte Wortliste in jeder Nachricht geprüft
🔗
Invite-Links
discord.gg und discord.com/invite werden blockiert
📢
Massen-Ping
Mehr als 5 User-Erwähnungen in einer Nachricht
🔠
Caps-Spam
>75% Großbuchstaben bei Nachrichten über 15 Zeichen
3
Verstöße → Warn
5
Verstöße → Mute
8
Verstöße → Kick
10
Verstöße → Ban
automod.py — live
# Nachricht geprüft
triggered = True
reason = "Schimpfwort"
 
# Nachricht löschen
await message.delete()
 
# Verstoß zählen
count = add_violation(guild, user)
# count = 3 → Verwarnung
 
# Konsequenz ausführen
await member.timeout(timedelta(minutes=10))
 
# DM + Log senden
await send_dm(member, embed)
await post_log(guild, log)
 
✓ Nachricht entfernt
✓ DM gesendet
✓ Log erstellt
Einrichtung

In 4 Schritten bereit

01

Bot einladen

Einladungslink öffnen, Server auswählen. Administrator-Berechtigung empfohlen.

02

Intents aktivieren

Im Developer Portal alle Privileged Intents aktivieren — besonders Message Content Intent.

03

Kanäle setzen

/setlog und /setwelcome für Logging und Welcome nutzen.

04

Fertig!

AutoMod läuft sofort. Mit /ticketsetup das Support-Panel erstellen.
