Alle Slash Commands

Alle Commands sind als Slash Commands verfügbar und sofort nach dem Einladen einsatzbereit.

🔨 Moderation
/banUser permanent vom Server bannen
/unbanGebannten User per ID entbannen
/kickUser vom Server kicken
/muteUser für X Minuten muten (Timeout)
/unmuteMute eines Users aufheben
/warnUser offiziell verwarnen
/warnsVerwarnungen eines Users anzeigen
/clearwarnsAlle Verwarnungen eines Users löschen
/clearBis zu 100 Nachrichten löschen
/slowmodeSlowmode im Channel setzen
ℹ️ Info
/helpAlle Befehle anzeigen
/botinfoInfos über den Bot
/userinfoInfos über einen User
/serverinfoInfos über den Server
/avatarAvatar eines Users anzeigen
🛡️ AutoMod
/addwordWort zur Blacklist hinzufügen
/removewordWort aus Blacklist entfernen
/blacklistAlle verbotenen Wörter anzeigen
/setconsequencesKonsequenzen einstellen (Warn/Mute/Kick/Ban)
/violationsVerstöße eines Users anzeigen
/resetviolationsVerstöße eines Users zurücksetzen
⚙️ Einstellungen
/setlogLog-Channel setzen + Logging aktivieren
/setwelcomeWelcome-Channel setzen
/testwelcomeWillkommensnachricht testen
/ticket-setupTicket-Panel im Channel erstellen
